A stylish and functional To-Do list built with **React** and **Bootstrap**, deployed on Vercel.

[![Live Demo](https://img.shields.io/badge/Live-Demo-green?style=for-the-badge&logo=vercel)](https://todo-react-eight-opal-32.vercel.app/)

## 🚀 Features

- ✅ Add, Edit, and Delete tasks
- ✅ Responsive UI with Bootstrap 5
- ✅ Cool green & mint gradient theme 🌿
- ✅ Deployed live using Vercel

## 🔗 Live Demo

👉 [https://todo-react-eight-opal-32.vercel.app/](https://todo-react-eight-opal-32.vercel.app/)

## 🛠️ Getting Started

Follow these instructions to set up the project locally.

### ✅ Prerequisites

- Node.js and npm installed on your machine

### 📦 Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/riddheshmisal/todo-react.git
````

2. Navigate into the project directory:

   ```bash
   cd todo-react
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Run the development server:

   ```bash
   npm start
   ```

The app will open in your browser at [http://localhost:3000](http://localhost:3000)

## 🧰 Tech Stack

* React (Class Components)
* Bootstrap 5
* Custom CSS
* Vercel (for deployment)
